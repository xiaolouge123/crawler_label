📝 测试步骤

  1. 配置设置： 右键插件图标 → 选项 → 输入 OpenAI API Key → 保存
  2. 选择内容： 点击插件图标 → 点击上方任一测试内容区域
  3. 处理内容： 在弹窗中输入建议的测试 Prompt → 点击生成
  4. 验证结果： 查看 AI 生成结果是否合理
  5. 测试导出： 点击"📁 导出 JSON"按钮 → 检查导出文件
  6. 检查日志： 按 F12 → Console → 查看是否有错误信息
