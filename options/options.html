📖 使用说明

  1. 配置 API Key: 在上方输入您的 OpenAI API Key 并保存
  2. 创建预设: 添加常用的 Prompt 模板，使用 {selection} 占位符
  3. 选择内容: 点击插件图标，然后点击网页上要提取的元素
  4. 处理内容: 在弹出窗口中选择预设或输入自定义 Prompt
  5. 获取结果: AI 处理完成后可复制结果或导出完整数据
